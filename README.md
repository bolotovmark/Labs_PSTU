## Вопросы
- "Что за посторонние файлы по типу 'CMake'?" - Начнём с того, что я пока что не смог разобраться с тем, как можно комфортно программировать, отлаживать код на Visual Studio Code.
(MinGW не патчиться. Не знаю с чем это связано, но решение буду искать до последнего) 
Поэтому я программирую на готовом решение от Jetbrains - CLion. Именно он и создаёт эти файлы отлажки/сборки. CMake - это мета-система сборки, которая использует сценарии, называемые CMakeLists, для создания файлов сборки для определенной среды (например, make-файлы на машинах Unix). Когда вы создаете новый проект CMake в CLion, файл CMakeLists.txt автоматически создается в корне проекта.
