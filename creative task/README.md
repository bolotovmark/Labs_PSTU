# Творческое задание 
## Особенности
- Версия Visual Studio 2022
- У задачи коммивояжера есть собственный автономный exe файл ``graph.exe``
- Калькулятор не релизнут (автономного exe нет), рекомендуется скачивать полную папку с проектом и открывать в ``...\Dates converter\bin\Debug\net6.0-windows\Dates converter.exe``
## Вопросы
- ``Решение не открывается в Visual Studio`` - скорее всего у вас VS более поздняя версия 
null- (решение написано на версии VS 2022)
- ``Проект с решением задачи коммивояжера не запускается из-за множества ошибок`` - скорее всего у вас не установлены дополнительные библиотеки opengl **freeglut в частности**
